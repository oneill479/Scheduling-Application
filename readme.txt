Title: Scheduling Application
Purpose: A scheduling desktop application that connects to a sql database and allows users to add, update, and delete appointments and customers.
Author: Caleb O'Neill
Contact: conei24@wgu.edu
Student Application Version: C195 QAM2
Date: 10/25/2021
IDE: IntelliJ Community 2021.1.3
JDK: Java SE 11.0.11
JavaFX: JavaFX-SDK-11.0.2

Directions:
	Step 1: Login with username and password
	Step 2: In the main menu you can choose customers, appointments or reports to go to each screen.
		- Customers allows the user to add, update, or delete customers
		- Appointments allows the user to add, update, or delete appointments
		- Reports shows the users different reports based on what the user is looking to find.

Additional Report: Created pie chart showing users by country

MySQL Connector: mysql-connector-java-8.0.26
